Introduction to Ruby on Rails (ROR-101)

Jun 18



Over the last couple of months, I have been learning web development, Ruby and Rails to be more specific. I wouldn't have fallen in love with Ruby if it wasn't for Rails.

So, WTH is Ruby on Rails?

Ruby on Rails is a backend (database) based, awesome, and really powerful web development platform designed/developed by David Heinemeier Hansson aka DHH.

“Rails is the most well thought-out web development framework I’ve ever used.And that’s in a decade of doing web applications for a living. I’ve built my own frameworks, helped develop the Servlet API, and have created more than a few web servers from scratch. Nobody has done it like this before.”

– James Duncan Davidson, Creator of Tomcat and Ant

Get your hands dirty...

To get started with Ruby on Rails

  • Install Ruby, Bundler, Ruby on Rails and all the goodies you need, I found this tutorial to be the best one so far.

  • Create a new Rails Project:-* 

    #Terminal
~> rails new my_app

  • cd into the directory

    ~> cd my_app

  • Fire up the rails server

    ~> rails server
#or
~> rails s

    Wallah your app is ready.

  • I am sure you don't above to be your homepage so lets go ahead and create a welcome index page.

    ~> rails generate controller welcome index
#or
~> rails g controller welcome index

#This command will create these many files
~> rails g controller welcome index
  create  app/controllers/welcome_controller.rb
   route  get 'welcome/index'
  invoke  erb
  create    app/views/welcome
  create    app/views/welcome/index.html.erb
  invoke  test_unit

  invoke  helper

  invoke    test_unit

  invoke  assets

  invoke    scss
  create      app/assets/stylesheets/welcome.css.scss

  • Now, if you refresh your page, you will still see "Welcome aboard" page but not the index page we just created.

  • To change the default root/homepage to the one we just created, head over to config/routes.rb, you will see something like this commented out, uncomment it.

    # root 'welcome#index'

    root 'welcome#index'

  • Reload your rails server, to reload it, cancel the running server (Command + C on Mac, and Ctrl + C on Windows/Ubuntu) and run the following command again. Did I mention, "reload your browser too!"

    ~> rails s

  • Damn, you just did it. Now head over to app/views/welcome/index.html.erb and start designing it.
